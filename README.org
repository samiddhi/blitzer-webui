* Overview

Blitzer Web UI provides a clean, minimalist web interface for the Blitzer CLI tool. It offers a user-friendly interface for processing text with various language and mode options using the underlying Blitzer command-line tool.

* Features

- Clean, minimalist design with digital minimalism principles
- Input and output text areas
- Language selection (Pali, Slovenian)
- Processing flag selection (lemmatize, frequency, context, prompt, source text)
- Toggle switches for additional options (--freq, --prompt, --src)
- Clear and process buttons
- Responsive design for different screen sizes

* Requirements

- Python 3.7+
- [[https://flask.palletsprojects.com/][Flask]] web framework
- [[file:../blitzer-cli][Blitzer CLI tool]] installed and in your PATH

* Installation

** Install Python dependencies:
#+BEGIN_SRC sh
cd ~/dev/blitzer-webui
pip install -r requirements.txt
#+END_SRC

** Make sure Blitzer CLI is installed:
#+BEGIN_SRC sh
pip install -e ~/dev/blitzer-cli
#+END_SRC

* Usage

** Start the server:
#+BEGIN_SRC sh
cd ~/dev/blitzer-webui
python server.py
#+END_SRC

The server starts on port 8080 by default.

** Open your browser and navigate to =http://localhost:8080=

** Use the interface:
   1. Enter your text in the input area
   2. Select language and flags using the controls
   3. Click "Process Text" to see results in the output area
   4. Use "Clear" button to reset the forms

** To use a different port:
#+BEGIN_SRC sh
PORT=3000 python server.py
#+END_SRC

* API

The application exposes a single API endpoint:
- =POST /api/blitzer= :: Process text with Blitzer CLI

Request body format:
#+BEGIN_SRC json
{
  "text": "Your input text here",
  "language": "pli",  // or "slv"
  "lemmatize": false,  // lemmatize words
  "freq": false,  // include frequency
  "context": false,  // include context
  "prompt": false,  // include prompt
  "src": false  // include source text
}
#+END_SRC

* Configuration

The server can be configured using the =PORT= environment variable:
#+BEGIN_SRC sh
PORT=3000 python server.py
#+END_SRC
